CSY1065 - Databases Design and Development

The purpose of this module is to apply the principles of database integrity in the design and development of database structures. This is a practical module that applies data modelling techniques to establish, modify and maintain database integrity, structures and associated components such as entities, relationships and attribute definitions. The module concentrates on the components of relational databases and the process of designing, translating and implementing effective database structures. Students develop user requirements into physical database structures using industry desirable software and languages such as Oracle and Structured Query Language (SQL). The module highlights the application of industry standards and good practice to optimise employability and performance.
